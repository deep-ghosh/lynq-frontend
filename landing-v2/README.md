# Landing V2

New landing page structure for Lynq DeFi Learning Platform.

## Structure

- **Home Section** - Hero/Landing area
- **About Section** - Product introduction
- **Student Experience** - What students learn and gain
- **Features Section** - Cool features and explanations
- **CTA Section** - Two actions:
  - Start Learning → `/learning` (Hands-on experience)
  - Launch App → Main app

## Development

```bash
npm install
npm run dev
```

Runs on http://localhost:3003
